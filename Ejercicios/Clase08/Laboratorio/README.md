# Showroom Taller Real CSS

Versión ampliada del laboratorio con fotografías reales enlazadas desde Wikimedia Commons.

## Importante

- No usa JavaScript.
- No incluye carpeta `js`.
- No incluye etiquetas `<script>`.
- No usa `@media` queries.
- La interactividad se hace con radios, checkbox, `:checked`, Grid Overlay, Flexbox, variables CSS y `@keyframes`.

## Fotos reales

Las imágenes están enlazadas desde Wikimedia Commons mediante `Special:Redirect/file/...` para mantener el proyecto liviano. Para verlas, abrí `index.html` con conexión a internet.

---

## 👨‍💻 Sobre este portafolio

Este repositorio reúne las prácticas, laboratorios y proyectos desarrollados durante el curso **Programación en Ambiente Web I**.

El objetivo principal es aplicar los fundamentos del desarrollo web moderno mediante la creación de soluciones funcionales, responsivas, accesibles y alineadas con buenas prácticas de la industria.

---

## 🚀 Enfoque del curso

Durante el desarrollo de este portafolio se trabajan temas relacionados con:

```txt
▸ Fundamentos de Internet y la Web
▸ HTML5 y estructura semántica
▸ CSS3 y diseño responsivo
▸ JavaScript moderno
▸ DOM, eventos e interacción web
▸ Consumo de APIs
▸ Git y GitHub
▸ Buenas prácticas de desarrollo
▸ Accesibilidad y experiencia de usuario
```

## 🎯 Objetivo personal

Fortalecer mis habilidades como desarrollador web mediante la construcción de proyectos que integren diseño, lógica, estructura, control de versiones e interacción con servicios externos.

Este portafolio representa mi avance técnico durante el curso y mi compromiso con escribir código más limpio, ordenado y profesional.

---

## 📌 Competencias desarrolladas

- Construcción de interfaces web responsivas.
- Aplicación de estándares modernos de HTML, CSS y JavaScript.
- Manejo básico de control de versiones con Git y GitHub.
- Manipulación del DOM para crear experiencias interactivas.
- Consumo de servicios externos mediante APIs.
- Organización de proyectos web con buenas prácticas.
